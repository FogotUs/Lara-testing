## Для запуска проекта

В корневой деректории проекта
```bash
Docker-compose up --build
```

Доступы к mariaDB  
````
DB_HOST=mariadb        
DB_PORT=3306    
DB_DATABASE=testing_db  
DB_USERNAME=laravel     
DB_PASSWORD=password
````

Сам Api: localhost:801

В репозитории также лежит простая [postman коллекция](testing.postman_collection.json)
